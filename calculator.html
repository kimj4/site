This is a little programming assignment I did for a job application that I want to show off because it is a good demonstration of my full-stack capabilityies.

The assignment was to build a calculator hosted on a website, that would keep a log of ten most recent results that would be visible by anybody that has the site open.

I made the calculator from raw HTML, CSS, Javascript, which made API calls to a DynaoDB backend whose operations were exposed through AWS API Gateway. The app queries the database for any changes periodically, and displays them in order.
